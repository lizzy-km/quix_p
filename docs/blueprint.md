# **App Name**: Vision Portfolio

## Core Features:

- Profile Display: Display profile information including name, contact details, and a brief introduction.
- Skills Showcase: Display technical and soft skills with visual representations of proficiency levels.
- Projects Gallery: Showcase completed projects with descriptions, images, and links to live demos or GitHub repositories.
- Experience Timeline: Present professional experience in a chronological timeline format, detailing roles, responsibilities, and achievements at each position.
- Education Overview: Provide an overview of educational background, including degrees, certifications, and relevant coursework.
- Contact Form: Allow visitors to send direct messages.
- Downloadable Resume: Provide a downloadable version of the resume in PDF format.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and creativity.
- Background color: Light grey (#F0F2F5) for a clean and modern backdrop.
- Accent color: Soft orange (#FFAB40) to highlight key elements and CTAs.
- Body and headline font: 'Inter' sans-serif, for a modern and neutral feel that maintains readability across all devices.
- Use simple, consistent icons from Mantine UI's library to represent skills and technologies.
- Employ a clean, responsive layout with clear sections for each part of the resume to improve readability.
- Use subtle animations (e.g., fade-in effects) on scroll to guide the user and create a dynamic user experience.
